# Summary

<em>German bellow</em>

----

<b>Arduino V3.0 (Nano) ATmega328P holder</b>
Small simple mount for the Arduino V3.0 (Nano) ATmega328P. The holder prevents the Arduino from getting a short circuit, to which the holder can be attached with four screws to other component groups.

----

<b>Arduino V3.0 (Nano) ATmega328P Halterung</b>

Kleine einfache Halterung für den Arduino V3.0 (Nano) ATmega328P. Die Halterung verhindert das der Arduino einen Kurzschluss erhält, zu dem lässt sich die Halterung mit vier schrauben an andere Bauteil Gruppen befestigen.
